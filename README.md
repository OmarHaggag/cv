# Omar Haggag's CV

This repository contains the LaTeX source code for Omar Haggag's curriculum vitae (CV).

## Contents

- `OmarHaggagCV.tex`: The main LaTeX file containing the CV content.
- `README.md`: This file, providing information about the repository.

## Building the CV

To compile the CV, you will need a LaTeX distribution (e.g., TeX Live, MiKTeX). You can compile the `OmarHaggagCV.tex` file using a LaTeX compiler like `pdflatex`.

```bash
pdflatex OmarHaggagCV.tex
```

This will generate a `OmarHaggagCV.pdf` file in the same directory.
